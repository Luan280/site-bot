Bot de Pagamentos no Telegram

Este bot foi desenvolvido para gerenciar pagamentos via PIX no Telegram.

Sobre

O bot permite que usuários adquiram planos de acesso e recebam automaticamente acesso a grupos privados após a confirmação do pagamento via Pix.

Termos de uso

Os pagamentos são processados pela API oficial do Mercado Pago. Nenhuma informação bancária dos usuários é armazenada pelo bot.
